プライバシーポリシー

「運動アニマル図鑑」（https://fit-animal.com。以下「本サイト」）は、ユーザーの皆さまに安心してご利用いただくため、以下のとおり情報の取扱いについて公表します。

1. 取得する情報

本サイトは、サービス提供および改善のため、以下の情報を取得することがあります。氏名・住所・電話番号・メールアドレス等の個人情報は取得しません。

  • 診断時の回答内容（端末内の localStorage に保存し、本サイトのサーバーには送信しません）
  • アクセスログ（IPアドレス、ブラウザ種別、リファラ、アクセス日時、ページURL）
  • 利用状況に関する情報（ページ閲覧、クリック・タップ位置、スクロール位置、画面サイズ）
  • Cookie および localStorage で発行する匿名識別子

2. 利用目的

  • 運動タイプ診断および関連機能の提供
  • 本サイトの利用状況の分析・UX 改善・不具合検出

3. 外部送信について（改正電気通信事業法 第27条の12 に基づく公表）

本サイトは、利用状況の分析のため Microsoft Clarity を利用しています。

  • 送信先事業者：Microsoft Corporation（米国）
  • 送信される情報：IPアドレス、ユーザーエージェント、リファラ、閲覧URL、画面サイズ、クリック・タップ位置、スクロール位置、Cookie・匿名識別子
  • 利用目的：本サイトの利用状況分析、ヒートマップ・セッションリプレイ生成による UX 改善
  • 送信先での取扱い：Microsoft プライバシーステートメント

4. 送信停止の方法

  • ブラウザの設定または拡張機能で *.clarity.ms へのアクセスをブロックする
  • ブラウザの設定から本サイトの Cookie・サイトデータを削除する
  • シークレット/プライベートウィンドウで利用する

5. 改定

本ポリシーは、法令変更や本サイト仕様の変更に応じて改定する場合があります。重要な変更は本ページにて告知します。
