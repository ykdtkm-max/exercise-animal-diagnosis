プライバシーポリシー

本ポリシーは、「運動アニマル図鑑」（https://fit-animal.com。以下「本サイト」）におけるユーザー情報の取扱いについて定めるものです。

1. 収集する情報

本サイトは、サービスの提供および改善のため、以下の情報を取得することがあります。

  • 診断時の回答内容（端末内の localStorage に保存。本サイトのサーバー外に送信されることはありません）
  • アクセスログ情報（IPアドレス、ブラウザ種別、リファラ、アクセス日時、ページURL 等）
  • 利用状況に関する情報（ページ閲覧履歴、クリック・タップ位置、スクロール位置、画面サイズ、操作ログ）
  • 匿名識別子（Cookie および localStorage で発行する乱数ベースのID。氏名・住所・連絡先等の個人を直接特定できる情報は含まれません）

2. 利用目的

  • 運動タイプ診断および関連機能の提供
  • 本サイトの利用状況の分析、UX 改善、不具合の検出・修正
  • 不正アクセス・不正利用の防止

3. 外部サービスへの情報送信について（外部送信規律に基づく公表）

改正電気通信事業法 第27条の12（外部送信規律）に基づき、本サイトが利用する外部サービスのうち、ユーザーの端末に蓄積された情報を外部に送信するものについて、以下のとおり公表します。

利用目的 本サイトの利用状況の分析、ヒートマップ・セッションリプレイ生成による UX 改善、エラー検出

4. オプトアウト方法

ユーザーは、以下の方法で本サイトでの情報取得を停止または制限できます。

  • Microsoft Clarity を停止する: ブラウザの拡張機能（uBlock Origin、Privacy Badger 等）でドメイン *.clarity.ms をブロックすることで、データ送信を停止できます。
  • Cookie および localStorage を削除する: ブラウザの設定から本サイト（fit-animal.com）の Cookie・サイトデータを削除すると、匿名識別子がリセットされます。
  • Do Not Track（DNT）を有効化する: ブラウザで DNT を ON にすると、当該シグナルに対応する送信先サービスでは追跡が制限されます。
  • プライベートブラウジング: シークレット/プライベートウィンドウで利用すると、終了時にすべての保存データが破棄されます。

5. Cookie および localStorage の利用

本サイトは利便性向上のため Cookie および localStorage を使用します。これには以下が含まれます。

  • 診断進捗（途中保存・復元）
  • 診断結果の保存
  • シェア進捗・SECRET UNLOCKED 既読フラグ
  • 解析用の匿名ビジターID（exerciseV2_visitor_uid）
  • Microsoft Clarity が発行する解析用 Cookie

6. 第三者への提供

本サイトは、法令に基づく場合を除き、ユーザー本人の同意なく取得した情報を第三者に提供しません。第3項に記載の外部サービスは「業務委託先」に該当し、各サービス事業者との契約および各社プライバシーポリシーに従って厳格に取り扱われます。

7. 個人情報の取扱い

本サイトは、現時点で氏名・住所・電話番号・メールアドレス等の個人情報を取得・保存しません。

8. お問い合わせ

本ポリシーまたは本サイトに関するお問い合わせは、本サイト運営者までご連絡ください。

9. 改定

本ポリシーは、法令変更や本サイトの仕様変更に応じて改定されることがあります。重要な変更がある場合は、本ページ上で告知します。
