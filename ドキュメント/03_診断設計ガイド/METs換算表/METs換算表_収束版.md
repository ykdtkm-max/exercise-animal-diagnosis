# 運動タイプ診断 METs換算マスターデータ（実践提案カタログ v3.0）

本ドキュメントは、診断結果において使用する「活動の実践提案カタログ」です。
**「どういう活動がどういう本質的な特性を持つか」を多角的に理解・整理することを目的としています。**

---

## ラベル体系リファレンス（10の特性軸）

各活動に付与するラベルの定義。テーブルでは「動機の本質」「継続させる仕組み」「参入ハードル」「達成感・報酬」「競争・評価構造」の5軸を使用。残り5軸（社交性/場所/日常性/強度/身体使用）は各カテゴリの説明文に記載。

### 軸A: 社交性（テーブル外・カテゴリ説明文に記載）
| 値 | 定義 |
| :--- | :--- |
| ひとり | 他者不要・完全な単独行動 |
| ペア・少人数 | 2〜5人程度の密な関係が前提 |
| 大人数・集団 | 多数が同じ場・熱量を共有する |

### 軸B: 場所・環境（テーブル外・カテゴリ説明文に記載）
| 値 | 定義 |
| :--- | :--- |
| 自宅・屋内 | 自分でコントロールできる固定空間 |
| 施設（ジム・スタジオ等） | 専用設備がある管理空間 |
| 屋外・街中 | 都市・商業環境 |
| 屋外・自然 | 公園・山・海・非人工環境 |
| 移動中 | 交通機関・徒歩・自転車での移動 |

### 軸C: 日常性・継続型（テーブル外・カテゴリ説明文に記載）
| 値 | 定義 |
| :--- | :--- |
| 毎日型 | 毎日のルーティンとして成立しうる |
| 週次型 | 週1〜2回のリズムで繰り返せる |
| 不定期型 | 気が向いたときや条件が揃ったとき |
| 非日常・単発型 | 計画・予約が必要なイベント系 |
| 季節限定型 | 特定の季節・時期のみ発生 |

### 軸D: 強度感（テーブル外・METs値から自動導出）
| 値 | METs目安 |
| :--- | :--- |
| 低（ほぼ気にならない） | 〜2.9 |
| 中（少し息が上がる） | 3.0〜4.9 |
| 高（汗ばむ・しんどい） | 5.0〜6.9 |
| 最高（全力・ハード） | 7.0〜 |

### 軸E: 動機の本質【テーブル列1】
**最も重要な軸。** 「なぜこの活動をやりたくなるのか」の根本的な欲求ドライバー。参考資料の「5つの欲求軸」とSDT（自律性・有能感・関係性）を統合。1活動に最大2つ付与。

| 値 | 定義 | 対応する欲求軸 |
| :--- | :--- | :--- |
| 達成・記録 | 数値・記録・マスタリーへの欲求 | Achiever / 有能感 |
| 社交・つながり | 仲間・帰属・コミュニティ欲求 | Socializer / 関係性 |
| 探索・自律 | 自由・発見・変化・新奇性への欲求 | Free Spirit / 自律性 |
| 没入・意味 | 世界観・美・ストーリー・没入への欲求 | Narrative / Purpose |
| 実利・効率 | ついで・一石二鳥・ゼロ摩擦への欲求 | Utilitarian / Utility |
| 競争・勝負 | 他者との比較・勝利・ランキング欲求 | Achiever競争面 |
| 表現・創作 | アウトプット・自己表現・作品欲求 | Narrative創作面 |
| 回復・整える | リセット・リラクゼーション・リカバリー欲求 | 精神的回復 |
| 義務・責任 | やらなければならない・他者のため | 外的調整 |

### 軸F: 継続させる仕組みの型【テーブル列2】
「何があれば続くか」という継続トリガーの本質。行動変容設計に直結。

| 値 | 定義 |
| :--- | :--- |
| アンカー統合型 | 既存の習慣（歯磨き・通勤）に自然に紐付く |
| 仲間・約束型 | 誰かとの約束・コミュニティがあると続く |
| イベント・締切型 | ライブ・大会・季節などの外部トリガーで動く |
| 記録・可視化型 | データや進捗が見えると続く |
| 自由・衝動型 | 気分や環境変化に乗じてやる・決まりがない方が続く |

### 軸G: 参入ハードル【テーブル列3】
「今日から始めるために何が必要か」を表す。行動変容の阻害要因の特定に直結。

| 値 | 定義 |
| :--- | :--- |
| 今すぐ（準備ゼロ） | 着替えも道具も不要・今この瞬間できる |
| 道具が必要 | 自転車・釣り竿・ヨガマット等の準備が必要 |
| 施設が必要 | ジム・スタジオ・アーケードへの移動が前提 |
| スキル習得が必要 | ある程度の練習・経験がないと楽しめない |

### 軸H: 達成感・報酬の種類【テーブル列4】
活動の「後」に得られる心理的報酬の型。「やってよかった」と感じる理由。

| 値 | 定義 |
| :--- | :--- |
| 数値・記録系 | 歩数・タイム・カロリー・重量が伸びた |
| 完成・変化系 | 部屋がきれいになった・料理が仕上がった・DIYが完成した |
| 上達・習熟系 | 技術・スキルが向上したと実感できた |
| 発散・解放系 | ストレスや疲れが消えた・気持ちが軽くなった |
| 没入・フロー系 | 時間を忘れた・完全に集中できた |
| 共感・一体感系 | みんなで体験した・場の熱量を共有した |
| 美的・審美系 | 景色が美しかった・体験が心地よかった |

### 軸I: 競争・評価の構造【テーブル列5】
「誰かに見られているか」「誰かと競っているか」の構造。

| 値 | 定義 |
| :--- | :--- |
| 競争なし | 純粋な個人体験・評価軸がない |
| 自己記録型 | 過去の自分との比較・自己ベスト更新 |
| 対人競争型 | 特定の他者と直接勝負する |
| チーム・協力型 | 仲間と共に同じ目標を目指す |
| 承認・観客型 | 他者に見せる・見てもらうことに意味がある |

### 軸J: 主な身体使用の種類（テーブル外・カテゴリ説明文に記載）
| 値 | 定義 |
| :--- | :--- |
| 全身協調 | 上下肢・体幹が連動する全身運動 |
| 心肺持久 | 有酸素・呼吸・循環系への負荷が主体 |
| 筋力・パワー | 特定筋群への負荷・瞬発力 |
| バランス・体幹 | 姿勢維持・軸・重心コントロール |
| 手先・巧緻性 | 繊細な操作・感覚・技巧 |
| 静止・呼吸 | ほぼ動かないが身体感覚に集中 |

---

> **消費カロリー算出公式**
> 消費カロリー（kcal） ＝ METs × 体重（60kgを想定） × 時間（1時間） × 1.05
> ※（1.0 METs ≒ 63 kcal）

> **免罪符ひとこと**
> ジョギング（7.0 METs）を基準に、各活動が「ジョギング何分相当か」を自動計算して付与。
