## ソースコード構成

```
src/app/exercise/
├── page.tsx                        # 診断メインページ（Welcome → Quiz → ResultReady → Result）
├── result/[id]/page.tsx            # シェアビュー（isSharedView=true）
├── _components/
│   └── ExerciseResultScreen.tsx    # 結果画面コンポーネント
└── _data/
    └── exerciseDiagnosis.ts        # 診断データ・ロジック

src/app/api/exercise-diagnose/
└── route.ts                        # POST API（計算 → DB保存 → JSON返却）
```

### 静的HTMLプロトタイプ（v2：4軸32問）

`prototype/exercise-v2/` に置いています（ログイン・API なし）。

- **起動**: `npx serve prototype/exercise-v2` を実行し、表示された URL を開く
- **ルーティング**: `#/`（ウェルカム）・`#/types`・`#/types/<CODE>`・`#/quiz`・`#/result`・`#/result/<CODE>`
