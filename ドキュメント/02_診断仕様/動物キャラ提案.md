# 16タイプ × 動物キャラ提案（各3候補）

**目的**: 16タイプのキャラデザインに活かすための動物選定  
**選定基準**: 動物の生態系を元に、一般人に馴染みがある動物であること  
**構成**: 各タイプに3候補（A/B/C）、各候補に選定理由・おすすめキャラ名3点

---

## Solo × Flow（衝動・感覚派・一人型）

---

### [SFAC] 気まぐれ散策者

**軸**: Solo / Flow / Assertive / Calm｜義務も目標もなく、気の向いたときだけ動く

#### 候補A：ネコ（Cat）

**選定理由**  
ネコは飼い主の都合ではなく自分の気分だけで動く。計画も記録もなく、ふと気が向いたときだけ体を動かすこのタイプと完全に一致。「呼んでも来ない」「なぜか夜中に走り出す」——義務感ゼロで生きる生態が重なる。

**一致ポイント**: 「義務も目標もない」「続けようと意識した瞬間に衝動が萎む」「一人を好む」


| キャラ名   | 方向性                    |
| ------ | ---------------------- |
| ふらり猫   | 目的なく気ままに動き回る、自由の象徴     |
| のんびり黒猫 | 神秘的でマイペース、誰にも縛られない存在感  |
| あてなし三毛 | どこへでも気分次第で向かう、方向音痴な自由人 |


#### 候補B：ナマケモノ（Sloth）

**選定理由**  
ナマケモノは必要なときだけ最小限だけ動く。動かないのがデフォルトで、動くのが例外。1日20時間眠り、移動するのは週に1度だけ——「運動を運動として意識しない」このタイプの理想の生き方に近い。

**一致ポイント**: 「続けようと意識した瞬間に動けなくなる」「計画・記録が体を重くする」「義務感ゼロ」


| キャラ名      | 方向性                     |
| --------- | ----------------------- |
| ぶらりナマケ    | 急がない、焦らない、気が向いたら動く      |
| のびのびナマケ   | 周囲の目など関係なく、ただ自分のペースで生きる |
| うとうとナマケモノ | 気づいたら動いていた、目的なき移動の達人    |


#### 候補C：キツネ（Fox）

**選定理由**  
キツネは好奇心の赴くまま単独で動き回る。誰にも縛られず、夜の気まぐれな散策者。計画はなく、気になったものに引き寄せられるだけ——「気になる道を通ってみたい」という微細な引力で動くこのタイプと重なる。

**一致ポイント**: 「微細な引力が出発点」「他者のペースに縛られない」「自由が動力の源」


| キャラ名    | 方向性                   |
| ------- | --------------------- |
| 夜道のキツネ  | 誰も見ていない夜に、気ままに散策する    |
| 気まぐれキツネ | ふらりと現れ、ふらりと消える自由な存在   |
| ふわふわ狐火  | 掴もうとすると消える、でも確かにそこにいた |

### [SFAD] 直感の冒険家

**軸**: Solo / Flow / Assertive / Drive｜衝動が来たとき、一人で全力を出す

#### 候補A：チーター（Cheetah）

**選定理由**  
チーターは計画ではなくスイッチが入った瞬間に全力疾走する。群れを作らず単独で狩り、終わったら長休憩する「全力→ゼロ」のサイクルが衝動型エンジンと重なる。同じ場所に留まらない習性も一致。

**一致ポイント**: 「直感的な衝動が出発点」「同じ活動の繰り返しに急速に飽きる」「次の体験への渇望」


| キャラ名    | 方向性                  |
| ------- | -------------------- |
| 疾風チーター  | 一瞬の閃きで全力発揮する最速の冒険者   |
| 衝動のチーター | 計画より感覚、止まるより走る       |
| 孤独なチーター | 誰とも組まず、自分の衝動だけで動き続ける |


#### 候補B：カワウソ（River Otter）

**選定理由**  
カワウソは滑り台で何度も遊ぶ探索好き。未知の水路を単独で泳ぎ回り、次の体験を常に探している。「また同じことか」とすぐ飽きる習性と、楽しければどこへでも行く行動力がこのタイプと重なる。

**一致ポイント**: 「次の体験・場所への欲求が更新され続ける」「一人での活動を好む」「体験ベースの動力」


| キャラ名     | 方向性                    |
| -------- | ---------------------- |
| 川のカワウソ   | 同じ川を二度と同じように流れない、常に次へ  |
| 流れるカワウソ  | 水の流れに身を任せながら、次の未知へ向かう  |
| ずぶ濡れカワウソ | 準備なんて関係ない、飛び込んだらなんとかなる |


#### 候補C：ハヤブサ（Peregrine Falcon）

**選定理由**  
世界最速の動物（急降下時速390km）。瞬間的な閃きで急降下し、次の瞬間には新しい空へ。計画なしの本能的な狩りと、同じ場所に留まらない自由なフライトが衝動型冒険家と重なる。

**一致ポイント**: 「直感が出発点」「新しい挑戦・場所・体験への欲求」「次の体験リストを更新し続ける」


| キャラ名    | 方向性                  |
| ------- | -------------------- |
| 閃光のハヤブサ | 一瞬の判断で全てを決める、迷わない冒険者 |
| 蒼空のハヤブサ | 空の高さから衝動のまま急降下する自由人  |
| 急降下の隼   | 終わった瞬間、もう次の目標に向かっている |

### [SFTC] ほっこり探索者

**軸**: Solo / Flow / Turbulent / Calm｜体より先に、心が動く

#### 候補A：ウサギ（Rabbit）

**選定理由**  
ウサギは環境が安全と感じられたときだけ活発に動く。外部の刺激に非常に敏感で、心が乱れると巣穴に引きこもる。自然の中をゆっくり動き回る姿は、心の状態が動力になるこのタイプと完全に一致する。

**一致ポイント**: 「心の状態にスイッチがある」「運動はフィットネスよりセラピーに近い」「心が重いと体がついてこない」


| キャラ名   | 方向性                      |
| ------ | ------------------------ |
| 静かうさぎ  | 穏やかで内省的、心と体が深く連動している     |
| 草原のウサギ | 周囲の変化に敏感で、安心できる場所でのびのびと  |
| ふんわり月兎 | やわらかく、自分のペースで月のように満ち欠けする |


#### 候補B：シカ（Deer）

**選定理由**  
シカは繊細な感受性を持ち、安全な自然環境でのみ伸び伸びと動く。心の状態や環境の変化に敏感で、強いプレッシャーには固まる。「動いた後に少し気持ちが変わった」という体験の質が継続の根拠になるこのタイプと重なる。

**一致ポイント**: 「義務感・競争よりも内側の感覚を丁寧に拾う」「一人の静かな時間で最も自然に運動できる」


| キャラ名  | 方向性                    |
| ----- | ---------------------- |
| 霧の迷い鹿 | 心が導く方向へ、霧の中をゆっくり歩く     |
| 静けさの鹿 | 騒がしい場所を離れ、自分だけの静寂を探す   |
| 森のシカ  | 周囲の空気を敏感に感じながら、そっと動き出す |


#### 候補C：コアラ（Koala）

**選定理由**  
コアラは特定の環境（ユーカリのある場所）が揃ったときだけ安心して動ける。ストレスに非常に敏感で、環境が合わないと動かなくなる。「心が整った後には自然に体も動き始める」このタイプの条件と重なる。

**一致ポイント**: 「心が重いときに義務感で動こうとしても体がついてこない」「動けない日は心の状態がそういう状態だったと受け取る」


| キャラ名    | 方向性                        |
| ------- | -------------------------- |
| ほっこりコアラ | 安心できる場所でだけ、そっと体を動かす        |
| 眠りのコアラ  | 休息と動きを自分のペースで決める、誰にも急かされない |
| 安心コアラ   | 心地よい環境が揃ったとき、自然と動き始める      |

### [SFTD] 全力一匹狼

**軸**: Solo / Flow / Turbulent / Drive｜スイッチが入ったとき、一人で誰より深く追い込む

#### 候補A：クマ（Bear）

**選定理由**  
クマは普段は静かで単独行動を好むが、スイッチが入ると圧倒的な力を発揮する。冬眠と活動期の「全力→ゼロ」のサイクルはこのタイプと重なり、群れに属さず自分の本能だけで動く生態が一致する。

**一致ポイント**: 「全力 or ゼロの二値」「スイッチが入った日の集中力は本物」「一人でいることを好む」


| キャラ名   | 方向性                   |
| ------ | --------------------- |
| 孤高の黒熊  | 誰にも頼らず、自分の力だけで全力を出す   |
| スイッチ熊  | 普段は静かでも、着火すると誰も止められない |
| 静かなるクマ | 群れを作らず、でも圧倒的な存在感を持つ一匹 |


#### 候補B：ヒョウ（Leopard）

**選定理由**  
ヒョウは単独・夜行性・木の上で潜んでいるが、スイッチが入ると一瞬で全力の狩りに移行する。一人で仕留め、一人で食べる。「中途半端な強度には価値を感じない」このタイプの美学と重なる。

**一致ポイント**: 「やるなら全力。それ以外に意味はない」「他者に縛られない完全な自由な環境」「スイッチが入った瞬間に動き出す」


| キャラ名  | 方向性                          |
| ----- | ---------------------------- |
| 夜のヒョウ | 誰も見ていない場所で、一人で最高の瞬間を作る       |
| 豹変の瞬間 | スイッチが入った瞬間、周囲が驚くほどの全力（豹＝ヒョウ） |
| 影のヒョウ | 静寂から突然現れる、一人完結の最高出力          |


#### 候補C：サイ（Rhinoceros）

**選定理由**  
サイは普段は草をゆっくり食べているが、スイッチが入ると時速50kmで突進し、何も止められない。単独行動を好み、普段と全力のギャップが最大級——「外から見ると『ムラがある』と映る」このタイプの特性と重なる。

**一致ポイント**: 「スイッチが入った日とそうでない日が別人のように異なる」「一人でいることを好む」「中途半端を嫌う」


| キャラ名    | 方向性                          |
| ------- | ---------------------------- |
| 一角のサイ   | 誰にも真似できない、サイの角のような唯一の突破力     |
| 鎧のサイ    | 分厚い皮膚（鎧）を持ち、スイッチが入ると誰も止められない |
| 静かな突進サイ | 普段は静かで、スイッチが入ると大地を揺らす        |

### [SPAC] ゆるり哲学者

**軸**: Solo / Plan / Assertive / Calm｜いつの間にか、ずっと続いている

#### 候補A：パンダ（Giant Panda）

**選定理由**  
パンダは毎日ほぼ同じルーティンで笹を食べ続ける。過激な動きはしないが確実に続ける。ストレスに強くマイペースで動じない。地味だが毎日こなす習慣の一貫性がこのタイプの哲学と重なる。

**一致ポイント**: 「決まったリズムで同じことを繰り返す」「地味な継続の哲学」「強い意志力なしに続けられる設計」


| キャラ名   | 方向性                  |
| ------ | -------------------- |
| のんきパンダ | 焦らず、マイペースで、でも確実に続ける  |
| 竹林のパンダ | 毎日同じ場所で、静かに深く動き続ける   |
| ゆるりパンダ | シンプルに、迷わず、ただ淡々と今日もやる |


#### 候補B：カメ（Tortoise）

**選定理由**  
「ウサギとカメ」の通り、カメは遅くても確実に前進し続ける継続の象徴。派手さも激しさも必要なく、「今日もやれた」という積み重ねが最大の充実感になる——このタイプの本質をそのまま体現している。

**一致ポイント**: 「派手な1週間より地味な1年が圧倒的な成果」「今日もやれたの積み重ねが最も強い自信の源」


| キャラ名   | 方向性                    |
| ------ | ---------------------- |
| 甲羅の哲人  | 重い甲羅を背負い、でも確実に一歩ずつ進む   |
| ゆっくりカメ | 焦らない、急がない、でも確実にゴールする   |
| 地道なカメ  | 目立たないが、長い目で見ると誰より遠くにいる |


#### 候補C：フクロウ（Owl）

**選定理由**  
フクロウは毎晩同じ木に止まり、毎晩同じルートをパトロールする習慣の動物。静かに観察し、自分のペースで行動する。「特別なことを組み込まず、生活に溶け込んだ動き」がこのタイプと重なる。

**一致ポイント**: 「既存の生活習慣に隣接させる」「気づいたらやっているレベルの習慣」「他者の都合に左右されにくい」


| キャラ名    | 方向性                      |
| ------- | ------------------------ |
| 静夜のフクロウ | 毎晩同じ場所で、静かに自分のルーティンを守る   |
| 梟の賢者    | 派手さゼロ、でも長い時間で最も賢い選択をしている |
| 毎晩のフクロウ | 同じ場所に、同じ時間に、ただ確実に現れる     |

### [SPAD] 孤高の挑戦者

**軸**: Solo / Plan / Assertive / Drive｜昨日の数字を、今日塗り替える

#### 候補A：ワシ（Eagle）

**選定理由**  
ワシは単独で空高く飛び、鋭い視力で目標を捉え正確に急降下する。常に技術を研ぎ澄まし、群れに属さず孤独に最高のパフォーマンスを追求する生態が、記録更新を動力にするこのタイプと完全に重なる。

**一致ポイント**: 「過去の自分との比較が動力」「一人でも高いモチベーションを維持」「停滞期が最大の危機」


| キャラ名  | 方向性                     |
| ----- | ----------------------- |
| 孤高のワシ | 群れを持たず、空の頂点を一人で目指す      |
| 蒼天のワシ | 数値と記録を武器に、昨日の自分を塗り替える   |
| 単騎のワシ | 誰かと競うのではなく、自分の記録だけを更新する |


#### 候補B：ハヤブサ（Peregrine Falcon）

**選定理由**  
世界最速の動物として「急降下時速390km」という数値の記録を持つ。精密な計算と観察による狩り——計測・記録・更新という三点セットがこのタイプの継続の前提条件と完全に一致する。

**一致ポイント**: 「測定可能な記録と進捗で駆動」「記録が更新されるとやめる理由がなくなる」「数値で証明できない成果は継続の根拠にならない」


| キャラ名    | 方向性                      |
| ------- | ------------------------ |
| 記録のハヤブサ | 数字で自分を定義する、測定なしには生きられない  |
| 極限のハヤブサ | 精密な計算の末に到達する、誰も見たことのない速度 |
| 時速390の隼 | 誰よりも高く、誰よりも速く、一人で限界を更新する |


#### 候補C：サメ（Shark）

**選定理由**  
サメは止まると死ぬ——常に前進し続けなければならない生き物。記録と目標なしに存在できない、孤独な追求マシン。「記録が途絶えた活動はやった実感を持ちにくく自然と遠のく」このタイプと重なる。

**一致ポイント**: 「記録があるから全力を出せる」「複数の指標を持ち常にどこかで更新できる数字を視野に」「停滞期に継続の意味を見失いやすい」


| キャラ名    | 方向性                     |
| ------- | ----------------------- |
| 止まれないサメ | 前進することだけが存在の意味、休息は次への充電 |
| 鮫の記録機   | 誰も知らない深さで、自分だけの数値を刻み続ける |
| 孤独なサメ   | 群れを作らず、記録だけを友に泳ぎ続ける     |

### [SPTC] ルーティン課長

**軸**: Solo / Plan / Turbulent / Calm｜完璧にやり切った日が、最高の報酬になる

#### 候補A：ビーバー（Beaver）

**選定理由**  
ビーバーは毎日同じ手順でダムを精密に設計・修繕し続ける。ルーティンの崩れに敏感で、型が乱れると直すまで気が済まない。反復の中に美学を見出すこのタイプの特性と生態が完全に一致する。

**一致ポイント**: 「曜日・時間・場所・順番にこだわり」「崩れると『やった気がしない』」「反復の中に深い洗練を見出す」


| キャラ名    | 方向性                    |
| ------- | ---------------------- |
| 几帳面ビーバー | 毎日ダムを点検し、1ミリのズレも見逃さない  |
| ダムのビーバー | 設計と修繕を繰り返し、完璧な型を追求し続ける |
| 棟梁ビーバー  | 反復の美学を持ち、型の完成度で自分を評価する |


#### 候補B：ハリネズミ（Hedgehog）

**選定理由**  
ハリネズミは毎晩同じルートを正確にパトロールする習慣の動物。ルーティンが崩れるとストレスを感じ、不安になる。一人で静かに、でも確実に型を守り続ける姿がこのタイプと重なる。

**一致ポイント**: 「曜日・時間・場所のこだわり」「ルーティン崩れ後の再起動コストが高い」「一人で反復することに美学を持つ」


| キャラ名     | 方向性                      |
| -------- | ------------------------ |
| 針山のハリネズミ | 全身の針のように鋭く、型のズレを見逃さない    |
| 丸まるハリネズミ | 乱れを感じると身を守るように丸まり、再起動を待つ |
| 夜道のハリネズミ | 毎晩同じ道を歩く、変わらぬ習慣の守り人      |


#### 候補C：アリ（Ant）

**選定理由**  
アリは毎日同じ道を正確にたどり、寸分の狂いもなく完璧な型を実行し続ける。一匹一匹が正確な役割をこなし、型が乱れることを許さない——このタイプが持つ「同じ時間に同じことをする美学」と重なる。

**一致ポイント**: 「同じ時間に同じことをする、それだけで整う」「完璧に実行できた日の充実感は独特」「型を反復することに価値を見出す」


| キャラ名     | 方向性                     |
| -------- | ----------------------- |
| 正確無比のアリ  | 毎日の型を0.1mmの誤差もなく実行する職人  |
| 行列のアリ    | 決まった道を、決まった順番で、寸分たがわず進む |
| 0.1mmのアリ | 誰も気づかない細部にこだわる、静かな完璧主義者 |

### [SPTD] 自己超越者

**軸**: Solo / Plan / Turbulent / Drive｜緻密な計画と、妥協なき追い込み

#### 候補A：トラ（Tiger）

**選定理由**  
トラは単独で完璧な狩りを追求し、決して妥協しない。計画的に獲物を追い、高い精度で実行する。孤独に強く、理想の実行に命をかける姿は「緻密な計画と妥協なき追い込み」のこのタイプと完全に重なる。

**一致ポイント**: 「計画の精度と実行の強度の両方を高水準で追い続ける」「計画が崩れると全部なかったことにしやすい」「身体より理想の計画を優先しがち」


| キャラ名   | 方向性                      |
| ------ | ------------------------ |
