## [SFAC] 気まぐれ散策者

**軸:** Solo / Flow / Assertive / Calm

**A2 キャッチコピー:**
義務も目標もなく、気の向いたときだけ動く。

**A3 タイプ概要:**
このタイプが最も自然に運動できるのは、義務でも目標でもなく、純粋な気分の衝動に乗ったときだ。「なんとなく歩きたい」「あの道を通ってみたい」という微細な引力が出発点になることが多い。「続けよう」と意識した瞬間にその衝動が萎む傾向があり、計画・記録・義務感が加わるほど体が重くなっていく。このタイプが最も長く運動を続けられるのは、それを「運動」と意識していないときだ。気になる場所をぶらつく、遠回りして帰る、行ったことのない道を歩く——目的のない移動の延長に、このタイプの体は十分に動いている。自由であることそのものが動力の源になっており、義務や計測の感覚が入り込んだとき、それまで自然に続いていた動きが止まることが多い。他者のペースや目標に縛られることなく、自分だけの感覚を頼りに動くとき、このタイプは最もよく運動できている。一日も動けない週があっても、それはこのタイプには「普通」のこと。続けるための仕組みを手放すことが、実は最も長続きする秘訣だ。

**A3T セクション見出し（タイプ概要）:**
目的のない移動

**A4 口癖:**
目的なんてない、なんか来ちゃった

**B1G 向いている運動:**
目的地も理由も持たない移動の延長に、このタイプの運動がある。散歩・のんびりしたサイクリング・気の向いたときのストレッチ・行ったことのない道を歩く散策が自然に合いやすい。「気になるから」「天気がいいから」「なんとなく」——そのような衝動のまま動き始められる活動がこのタイプに最もよく機能する。記録も報告も約束も必要なく、気が向いたときに始められる自由な動き方が、このタイプの本来の運動スタイルだ。

**B1GT セクション見出し（向いている運動）:**
気分のまま動く

**B1B 向いていない運動:**
年会費制のジム通い・毎日の記録管理・タイムアタック型の運動は、義務感が生まれた瞬間にこのタイプのエンジンを止めてしまいやすい。歩数を記録する・誰かに進捗を報告する・「今週○回やる」と宣言するといった管理の仕組みは、それまで自然にできていた運動をかえって止める原因になりやすい。決まった曜日に参加しなければならないスポーツや、チームの都合に合わせる必要がある活動は、このタイプには負荷になりやすい。

**B1BT セクション見出し（向いていない運動）:**
管理が動きを止める

**C1 継続のコツ:**
「継続しよう」という意識を手放すことが、このタイプには最も機能しやすい。遠回りして帰る・気になる道を通ってみる・天気がいいからとりあえず外に出る——そういった小さな出発を積み重ねるだけでよい。記録アプリを入れない、誰かに報告しない、目標を設定しないことが継続の前提条件になる。「最近運動してる？」と聞かれることもプレッシャーになりやすいため、自分の中だけで続けていることが、このタイプの理想の環境だ。次に歩きたいルートや気になる場所を頭の中にメモしておくと、気分が乗ったときに自然に体が動き出しやすくなる。

**C1T セクション見出し（継続）:**
「続けよう」をやめるコツ

**D3 相性タイプ:**

- [good] code: SFTC / name: ほっこり探索者 / reason: 同じソロ×Flow×Calm系。義務感なしに穏やかに運動する姿勢が一致する。強制も競争もなく、気が向いたときにゆるく動ける関係が自然に長続きする。

## [SFAD] 直感の冒険家

**軸:** Solo / Flow / Assertive / Drive

**A2 キャッチコピー:**
衝動が来たとき、一人で全力を出す。

**A3 タイプ概要:**
このタイプの運動エンジンは、「次に行く場所」「次に試すこと」という具体的なビジョンによって駆動されることが多い。「今日なんか燃えてきた」という直感的な衝動が出発点になりやすく、計画よりも本能に従って動く傾向がある。同じ活動の繰り返しに急速に飽きを感じ、新しい挑戦・場所・体験への欲求が常に更新され続けることが継続の前提になっている。一人での活動を好み、他者のペースに縛られない自由な環境でこそ最大のパフォーマンスを発揮することが多い。固定されたルーティンはこのタイプの運動意欲を徐々に奪っていく。このタイプにとって「続ける」という概念は、同じことを繰り返す継続ではなく、体験の種類が更新され続ける連続に近い。終わった体験は次の体験の踏み台であり、同じことに戻ることへの強い抵抗感があるのはそのためだ。「また同じことをしている」と感じた瞬間は、次のステップを探し始めるサインとして積極的に受け取るとよい。

**A3T セクション見出し（タイプ概要）:**
次の体験への渇望

**A4 口癖:**
計画とかないけど、まあ行ったら何とかなるでしょ

**B1G 向いている運動:**
登山・ボルダリング・サーフィン・HIIT・格闘技・ロングライドのように、強度が高く体験として記憶に残る活動が自然に合いやすい。「次の体験リスト」に追加したいと思えるような活動がこのタイプのエンジンに火をつける。一人で計画なしに山や海へ向かうことも、このタイプにとっては十分なフィットネスになりうる。新しい場所・新しい挑戦・初めての体験というキーワードが動力になる。

**B1GT セクション見出し（向いている運動）:**
衝動が選ぶ活動

**B1B 向いていない運動:**
毎週同じジムに通う固定スケジュール型・毎朝決まった時間に行うルーティン強制型・チームの都合に合わせ続ける集団スポーツは、このタイプには馴染みにくい。繰り返す構造そのものが苦手なため、同じ活動を同じ強度・同じ場所でやり続けることはモチベーションを急速に失わせる。タイムや回数の細かい記録管理も合いにくく、「まだ続けなきゃ」という義務感が運動を重くする。

**B1BT セクション見出し（向いていない運動）:**
同じことの繰り返しが動力を消す

**C1 継続のコツ:**
「次に行きたい場所・試したい活動」のリストを常に更新しておくことが最も機能する継続の仕組みだ。次の体験の候補がなくなった瞬間にモチベーションが急落するため、「今の体験が終わる前に次を決める」習慣をつけるとよい。同じルートや同じメニューが繰り返しになってきたと感じたら、すぐに新しい活動に切り替えるのが自分に合った対処法だ。体験ベースの動力を活かすため、SNSやアウトドア系メディアで「次行きたい場所」を眺める時間を設けておくことも有効だ。「また同じことをしている」と感じた時点で、次のステップを探し始めるサインとして受け取るとよい。

**C1T セクション見出し（継続）:**
次の体験を先に決めるコツ

**D3 相性タイプ:**
- [good] code: SPAD / name: 孤高の挑戦者 / reason: 同じ高強度ソロ型。SFADの衝動にSPADの記録と計画が加わり、衝動を成果に変えられる組み合わせ。目標に向かう熱量が共鳴し、互いを刺激しながら成長できる。
- [good] code: GFAD / name: お祭り遊牧民 / reason: 同じFlow×Drive系。「次どこ行く？」という会話がそのまま運動計画になる。互いの体験欲求が増幅し合い、新しい挑戦へのエネルギーが持続しやすい。

- [bad] code: GPTC / name: チームの守り神 / reason: チームへの責任・決まった練習日・進捗管理はSFADの気まぐれなエンジンと相容れない。義務感が生まれた瞬間に熱が冷めるSFADにとって、消耗しやすい関係になりやすい。

## [SFTC] ほっこり探索者

**軸:** Solo / Flow / Turbulent / Calm

**A2 キャッチコピー:**
体より先に、心が動く。

**A3 タイプ概要:**
このタイプが運動したくなるとき、そのスイッチは心の状態にあることが多い。気持ちが重いとき、頭が整理されていないとき、なんとなく外の空気が吸いたいとき——その微細な内側の変化が動き出す合図になりやすい。義務感や数値目標ではなく、「動いた後に気持ちが変わった」という体験の質が継続の根拠になっている。このタイプにとって運動はフィットネスよりもセラピーに近く、心と体が深く連動しているため、心が重いときに義務感で動こうとしても体がついてこないことが多い。外からの刺激や競争よりも、自分の内側の感覚を丁寧に拾いながら行動するタイプで、一人の静かな時間の中で最も自然に運動できる。心の状態に波があるぶん、運動できる日とそうでない日の落差が大きくなりやすいのも、このタイプのひとつの特徴だ。「運動できない日があるのは意志が弱いからじゃない」——そう自分に伝えられることが、長く続けるための一番大切な認識だ。

**A3T セクション見出し（タイプ概要）:**
心と体のつながり

**A4 口癖:**
動くとなんか、頭が静かになる

**B1G 向いている運動:**
自然の中での瞑想ウォーク・自宅ヨガ・公園のぼーっと歩き・ゆっくりした水泳・就寝前の軽いストレッチが自然に合いやすい。「動いた後に少し気持ちが変わった」という感覚が得られる活動が、このタイプにとっての本当の意味での運動になる。心への負荷が少なく、自分のペースで進められ、終わったあとに「整った」と感じられるものを選ぶとよい。一人で静かに体と向き合える環境が最も機能する。

**B1GT セクション見出し（向いている運動）:**
心を整える活動

**B1B 向いていない運動:**
大人数でワイワイするグループレッスン・タイムや回数を競う運動・高強度の追い込み系エクササイズは、このタイプの心に負荷をかけやすい。心が重い日や疲れている日に義務として運動しようとすることも、かえって消耗を深める結果になりやすい。強度が高くなるほど・プレッシャーが加わるほど「しんどいもの」になっていく傾向があるため、競争・記録・達成目標を前面に出した活動は避けるほうが続けやすい。

**B1BT セクション見出し（向いていない運動）:**
心に負荷がかかる環境

**C1 継続のコツ:**
心が重い・頭が整理されていない・外の空気が吸いたい——そういう微細なサインを運動のきっかけとして積極的に活用するとよい。「気持ちを変えたいから動く」という動機を前面に出した設計が機能しやすい。完璧な状態で始めようとせず、「5分だけ歩く」「ストレッチだけする」というように量を極限まで小さく設定することが継続のカギになる。運動できなかった日の自己嫌悪は不要で、動けない日は「今日の心の状態がそういう状態だった」と事実として受け取るだけでよい。心が整った後には自然に体も動き始めるため、動けない時期に無理に習慣化しようとしないことが長期継続の条件だ。

**C1T セクション見出し（継続）:**
心のサインを動く理由にする

**D3 相性タイプ:**

## [SFTD] 全力一匹狼

**軸:** Solo / Flow / Turbulent / Drive

**A2 キャッチコピー:**
スイッチが入ったとき、一人で誰より深く追い込む。

**A3 タイプ概要:**
このタイプの運動は「全力 or ゼロ」の二値に近い。スイッチが入った日の集中力と追い込みは本物で、一人で静かに、しかし内側では誰より熱く燃えていることが多い。「やるなら本気でやりたい」という美学があり、中途半端な強度の活動には価値を感じにくい傾向がある。スイッチが入らない日は運動意欲が湧かず、継続の波が大きい。毎日コツコツ続けるという継続モデルよりも、全力を出せた日の体験が積み重なることで記録が形成されていくのが、このタイプの特徴だ。スイッチが入っていないときの自分と、入っているときの自分が別人のように異なり、外から見ると「ムラがある」と映ることが多い。でも、それはこのタイプの燃料の性質であって、意志力の問題ではない。一人でいることを好み、他者のペースに縛られない環境での活動が本領を発揮する条件になる。

**A3T セクション見出し（タイプ概要）:**
全力かゼロか

**A4 口癖:**
やるなら全力。それ以外に意味はない

**B1G 向いている運動:**
ランニング・格闘技・本格的な筋トレ・ハードな登山・ボルダリングのように、自分の意志だけで完結できる高強度の一人活動が自然に合いやすい。「スイッチが入った」と感じた瞬間に即座に動き出せる活動が最もよく機能する。全力を出せる環境——プレッシャーなしに自分の限界を試せる場——がこのタイプの理想の運動環境だ。

**B1GT セクション見出し（向いている運動）:**
全力で入れる活動

**B1B 向いていない運動:**
週3回必ず参加を前提にしたスポーツクラブ・グループでのんびり雑談しながらするレッスン・毎回記録をつけて更新を義務化する管理型プログラムは、このタイプには合いにくい。中途半端な強度の活動は価値を感じにくく、義務として動かされる仕組みがスイッチをさらに遠ざける。参加しなければならないコミットメントが積み重なることで、運動そのものが重荷になりやすい。

**B1BT セクション見出し（向いていない運動）:**
義務とゆるさがスイッチを消す

**C1 継続のコツ:**
スイッチが入ったときに即座に動き出せる環境を常に整えておくことが最も重要だ。ジムは家の近く、ランニングシューズはすぐ手に取れる場所——着火から動き始めるまでの障壁を極限まで下げておくとよい。スイッチが入りやすい状況を観察して記録しておくと（「プライドが刺激されたとき」「心に余裕があるとき」など）、そのパターンを意図的に作れるようになる。毎日の継続を前提とした設計は合わないため、「スイッチが入った日に全力」というリズムを継続の単位として定義し直すとよい。スイッチが入らない日が続いても、それは継続の失敗ではない。

**C1T セクション見出し（継続）:**
着火環境を整えるコツ

**D3 相性タイプ:**
- [good] code: SFAD / name: 直感の冒険家 / reason: 同じソロ×Flow×Drive系の高強度型。どちらも「スイッチが入ったら全力」という価値観を共有する。干渉せず互いの熱量を尊重しながら並走できる自然な関係。
- [good] code: GFTD / name: 熱狂伝道者 / reason: 同じFlow×Turbulent×Drive系。熱量のベクトルが一致しており、互いのスイッチを入れ合う刺激がある。GFTDの外向きの熱量がSFTDの着火剤になることも多い。
- [bad] code: GPAC / name: コミュニティ長老 / reason: 「みんなで長く安定して続ける」というGPACの価値観はSFTDの「全力かゼロ」という姿勢と根本から衝突する。ペースを合わせる圧力がSFTDの運動意欲を消す。
- [bad] code: SPTC / name: ルーティン課長 / reason: 毎日の習慣と記録を重視するSPTCの継続観はSFTDにとって最も縛られる構造。「スイッチが入らない日でも動かされる」形式が、SFTDを徐々に追い詰めていく。

## [SPAC] ゆるり哲学者

**軸:** Solo / Plan / Assertive / Calm

**A2 キャッチコピー:**
